Pill Care

Shixuan Wu


Description (Tentative):

A pill-taking reminder consists of multiple stickers and a box. Family members can help the patients set up the alarm by recording their voices and sticking the sticker to the bottom of the pill bottle. They also need to set up the alarm time per day.

During the time the user needs to take pills, the alarm will sound indicating which sticker/pills they need to take with the pre-recorded sounds from their family members/physicians. The user then can take the bottle next to the reminder, if itâ€™s the correct one, the reminder will state that it's correct and the amount the user should take each time; if it's not, the reminder will ask the user to try another bottle.

Laws of UI Design and examples

Jacob's Law


  When I try to switch between Mac operating system and Windows operating system, I notice the similarity in their setting window. There are options with similar names and layout, such as same position of the account, Notification, Languages and other options.


Law of Proximity


  In the Google Docs, I notice the options closed together ususally serves similar purposes. The options for different alignemnts are closed together. Also, under the Format tab, different modifications to our page format can be found here, such as spacing and bullets numbering.


Millers Law


  In the Google or any other web browsers, I notice that we can bookmark the useful pages so that it will create shortcut on the top and we can click that to access that webpage with ease. As people are forgetable, bookmarking useful websites, such as the wesbite of the scientific calculator or the interesting videoes you just watched, can remind people to revisit them at a later time.
