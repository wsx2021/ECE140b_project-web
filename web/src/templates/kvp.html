Description (Tentative):

A pill-taking reminder consists of multiple stickers and a box. Family members can help the patients set up the alarm by recording their voices and sticking the sticker to the bottom of the pill bottle. They also need to set up the alarm time per day.

Product Vision
Alarm
the alarm will sound indicating which sticker/pills they need to take with the pre-recorded sounds from their family members/physicians.
Screen
screen that can be attached to the Pill Care to display large font text for hearing-impaired elders
App
app which sends notification to the user and the family whether pills need to be taken, and when the pills are taken by the patient.
Customerized Voice recorder
Voice of your family memeber will be the reminders which will state that it's correct and the amount the user should take each time; if it's not, the reminder will ask the user to try another bottle.
